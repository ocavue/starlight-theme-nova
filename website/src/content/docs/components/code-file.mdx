## `<CodeFile>` Props


The `<CodeFile>` component accepts the following props, as well as all other [`<Code>` component props](/components/code):


### `path`

**required**

**type:** `string`

The file path relative to the project root.

### `title`

**type:** `string`

The title of the file. If not provided, the file name will be used.

### `lang`

**type:** `string`

The language of the file. If not provided, the file extension will be used.
