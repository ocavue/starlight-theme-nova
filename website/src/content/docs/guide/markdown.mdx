## Code Block with Title

Input:

<Code code={CodeTitleCode} lang="md" />

Output:

<CodeTitle />

## Code Tabs

Input:

<Code code={CodeTabsMarkdownCode} lang="md" />

Output:

<CodeTabsMarkdown />

## Line Highlighting in Code Blocks

Input:

<Code code={ShikiHighlightLineCode} lang="md" />

Output:

<ShikiHighlightLine />

## Word Highlighting in Code Blocks

Input:

<Code code={ShikiHighlightWordCode.replace('[!', '[\\!')} lang="md" />

Output:

<ShikiHighlightWord />

## Diff Highlighting in Code Blocks

Input:

<Code code={ShikiDiffCode} lang="md" />

Output:

<ShikiDiff />

## Twoslash

Nova supports [Twoslash](https://twoslash.netlify.app/) out of the box. Hover over the code block to see the inline type.

Input:

<Code code={TwoslashCode} lang="md" />

Output:

<Twoslash />
