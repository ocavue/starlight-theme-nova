## `<CodeTabs>` Props

### `syncKey`

**type:** `string`

A key used to keep multiple tab groups synchronized across multiple pages.

## `<CodeTabItem>` Props

The `<CodeTabItem>` component accepts the following props, as well as all other [`<Code>` component props](/components/code):

### `label`

**required**

**type:** `string`

The label to display in the tab.
