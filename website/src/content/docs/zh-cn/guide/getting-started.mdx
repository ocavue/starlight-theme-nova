一个基于 [Starlight](https://starlight.astro.build/zh-cn) 的现代 / 漂亮的主题.

## 先决条件

你需要先创建一个 Starlight 网站.

如果你还没有, 你可以通过 ["开始使用"](https://starlight.astro.build/zh-cn/getting-started) 指南在 Starlight文档中创建一个.

## 安装

import { Steps } from '@astrojs/starlight/components'
import { CodePackageManagers } from 'starlight-theme-nova/components'

<Steps>

1. Starlight Nova 主题是一个 Starlight [插件](https://starlight.astro.build/zh-cn/reference/plugins/), 你可以使用自己喜欢的包管理器安装

   <CodePackageManagers packages="starlight-theme-nova" />

2. 在 `astro.config.mjs` 中 [配置 Starlight 插件](https://starlight.astro.build/zh-cn/reference/configuration/#plugins).

   ```ts twoslash
   // astro.config.mjs
   import starlight from '@astrojs/starlight'
   import { defineConfig } from 'astro/config'
   import starlightThemeNova from 'starlight-theme-nova'

   export default defineConfig({
     integrations: [
       starlight({
         plugins: [
           starlightThemeNova(/* options */), // [!code ++]
         ],
         title: 'My Docs',
       }),
     ],
   })
   ```

   在下面的 [Options](#options) 部分获取可用项.

3. [启动开发服务器](https://starlight.astro.build/zh-cn/getting-started/#start-the-development-server) 以预览主题.

</Steps>

就是这样! 现在你应该可以看到 Starlight Nova 主题已经应用到你的 Starlight 网站.

## 选项

### `nav`

你可以通过将一些对象传递到 'nav' 选项以在顶部导航栏添加一些可选的导航项.

```ts twoslash
import starlightThemeNova from 'starlight-theme-nova'
// ---cut---
starlightThemeNova({
  nav: [
    { label: 'Quick start', href: '/guide/getting-started' },
    { label: 'External link', href: 'https://example.com' },
  ],
})
```

如果需要支持多语言, 你可以将每个不同语言环境的值传递给 `label` 和 `href` 对象。当使用 object 对象时, 键值必须是相应语言的 BCP-47 标签 (例如 `en`, `ar`, 或者 `zh-CN`) 或 `root` 键来设置默认值：

```ts twoslash
import starlightThemeNova from 'starlight-theme-nova'
// ---cut---
starlightThemeNova({
  nav: [
    {
      label: {
        en: 'Quick start',
        'zh-CN': '快速开始',
      },
      href: {
        en: '/guide/getting-started',
        'zh-CN': '/zh-cn/guide/getting-started',
      },
    },
  ],
})
```

## 与 Tailwind CSS 一起使用

你可以选择将 Tailwind CSS 与 Starlight Nova 主题一起使用。为此，你需要：

<Steps>

1. 安装 Tailwind CSS 及其依赖项：

   ```bash
   npm install tailwindcss @tailwindcss/vite
   ```

2. 创建 `./src/styles/global.css` 文件并添加以下内容：

   ```css
   /** ./src/styles/global.css */
   @import 'tailwindcss';
   @import 'starlight-theme-nova/tailwind.css';
   ```

3. 配置你的 Astro 配置文件：

   ```ts
   // astro.config.mjs
   import starlight from '@astrojs/starlight'
   import tailwindcss from '@tailwindcss/vite' // [!code ++]
   import { defineConfig } from 'astro/config'
   import starlightThemeNova from 'starlight-theme-nova'

   export default defineConfig({
     vite: {
       plugins: [tailwindcss()], // [!code ++]
     integrations: [
       starlight({
         plugins: [starlightThemeNova()],
         title: 'My Docs',
         customCss: ['./src/styles/global.css'], // [!code ++]
       }),
     ],
   })
   ```

</Steps>
